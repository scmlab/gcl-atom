# gcl-atom

## Commands

| Keymap                             | Command                                 |
|-----------------------------------:|:----------------------------------------|
| <kbd>C-c</kbd> <kbd>C-x</kbd>      | activate/deactivate the package         |
| <kbd>Cmd-s</kbd> or <kbd>C-s</kbd> | save and reload the file                |
| <kbd>C-c</kbd> <kbd>C-c</kbd>      | refine a hole                           |
